# Experimentarium
### Название проекта
Experimentarium - среда для постановки физических экспериментов
### Авторы проекта
Путилин Даниил
### Описание идеи
Experimentarium - удобоне приложение для визуалицации экспериментов и опытов по физике.
Можно использовать для проведения лабораторных работ, проверки теорий
или постановки экспериментов, которые требуют большого количества
ресурсов.
### Описание технологий
Почти весь код напсиан на Python.
Используемые библиотеки:
* PyQt6
* math
* sqlite3
* re
* hashlib
* io
* sys

Для создания шаблонов окна использовалось приложение QtDisigner
### Описание реализации
Всё приложение описано с помощью ООП:
* MyMainWindow(QWidget) - главное окно (лаборатория)
* AuthorizationWindow(QWidget) - окно авторизации
* SignUpWindow(QWidget) - окнорегистрации
* ExperimentWindow(QWidget) - основное окно экспериментов
    + BallisticExpWindow(ExperimentWindow) - окно эксперимента по баллистике
    + MKT_ExpWindow(ExperimentWindow) - окно эксперимента типа МКТ
